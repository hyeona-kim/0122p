# SpringBoot-Project LMS


* mvc패턴 평가관가관리 시스템 

## 📃 프로젝트 소개 
학생 관리 웹 시스템으로 온라인으로 학생들의 성적,출석 등을 관리하고 교육 전반에 걸쳐 관리자나 담당 강사가 학습을 평가하고 관리에 이르기까지 전반적인 과정들을 통합적으로 운영 및 관리하기 위한 웹 시스템입니다.
